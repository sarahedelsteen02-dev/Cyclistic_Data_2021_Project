# Om mitt første dataanalyse prosjekt
Dette prosjektet er en del av Google Data Analytics Beginner Certificate og mitt første dataanalyseprosjekt. Jeg har analysert en fiktiv casestudie om sykkeldelingsselskapet Cyclistic i Chicago ved hjelp av en strukturert dataanalyseprosess for å identifisere innsikter og støtte datadrevne beslutninger. Mer informasjon om prosjektet finner du i dokumentet!
